% 1. CITAÇÃO NUMÉRICA BÁSICA:
%    Comando: \cite{chave}
%    Resultado: [1]
%    Exemplo: "Estudos recentes demonstram a eficácia do método \cite{autor2023}."
%
% 2. CITAÇÃO COM AUTOR NA SENTENÇA:
%    Comando: \citeonline{chave}
%    Resultado: Autor (ano) ou Autor [1] (depende da configuração)
%    Exemplo: "Segundo \citeonline{autor2023}, a metodologia é adequada."
%
% 3. MÚLTIPLAS CITAÇÕES:
%    Comando: \cite{chave1,chave2,chave3}
%    Resultado: [1, 2, 3]
%    Exemplo: "Diversos autores concordam \cite{autor2023,autor2022,autor2021}."
%
% 4. CITAÇÃO COM PÁGINA ESPECÍFICA:
%    Comando: \cite[p.~25]{chave}
%    Resultado: [1, p. 25]
%    Exemplo: "Como afirmado \cite[p.~25]{autor2023}, o resultado foi positivo."
%
% 5. CITAÇÃO INDIRETA/PARÁFRASE:
